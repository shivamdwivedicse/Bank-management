# 🏦 Bank Management System (Python Project)

The **Bank Management System** is a command-line based Python application designed to simulate basic banking operations. This project demonstrates the practical implementation of core programming concepts such as **Object-Oriented Programming (OOP), file handling, data structures, and user authentication**.

The system allows users to create and manage their bank accounts efficiently, with all data being stored persistently in a JSON file.

---

## 🚀 Features

- 🔐 **Account Creation**
  - Users can create a new bank account with basic details.
  - Unique account number is automatically generated.
  - Age and PIN validation ensures secure access.

- 💰 **Deposit Money**
  - Users can deposit money into their account (with limit control).

- 💸 **Withdraw Money**
  - Withdrawal is allowed only if sufficient balance is available.

- 📄 **View Account Details**
  - Secure login using Account Number and PIN to view details.

- ✏️ **Update Account Information**
  - Users can update name, email, and PIN.

- ❌ **Delete Account**
  - Users can permanently delete their account with confirmation.

## 💡 How It Works

- All user data is stored in a `data.json` file.  
- Each operation (create, update, delete, etc.) modifies the JSON database.  
- The system authenticates users using **Account Number + PIN** before any transaction.
